### 补充说明

- **外部调用**：适用于宿主机（如你当前的 RockyLinux）或宿主机外的其他机器通过 `192.168.80.101` 或 IP 访问。
- **容器内部**：适用于**同一 Docker 网络内**的其他容器通过服务名（如 `milvus-standalone`）访问。
- **注意事项**：
  - Milvus 使用的是 gRPC 协议（端口 19530），不是 HTTP。
  - ClickHouse 有两个端口：`8123`（HTTP API）和 `9000`（Native 协议）。
  - Neo4j 也有两个端口：`7474`（浏览器界面）和 `7687`（Bolt 协议，用于应用程序连接）。
  - Redis 有密码保护，连接时记得加上 `AUTH redis_password_2026`
